## 项目介绍
基于springboot+vue的医院管理系统：前端 vue、bootstrap、coreui，后端 maven、springmvc、spring、mybatis、redis，角色分为管理员、医生、医技医生、药房、挂号；集成挂号、检查申请、成药处方、处置申请等功能于一体的系统。
